# Fysisk Biokemi course utilities

This package contains Python utilities used by the Fysisk Biokemi course
material, including datasets, plotting helpers, and interactive widgets.

The package is installed into the course rendering environment from the
repository root through Pixi. For Google Colab or notebook use, install the
package with the appropriate optional dependencies, for example:

```sh
pip install "fysisk-biokemi[colab] @ git+https://github.com/au-mbg/fysisk-biokemi.git#subdirectory=course-utils"
```

The import package name is `fysisk_biokemi`.
